# Woodworking Helper
A simple woodworking utility app built with Kotlin and Jetpack Compose.

## Features

**5 Cut Method Calculator**
Calculates the fence adjustment required to square a cross-cut sled on a table saw using the 5 cut method.

**Inches to MM Converter**
Converts imperial measurements to metric.

## Formulas

**5 cut method:**
`Adjustment = ((A - B) / 4) × (Pivot Distance / Cut Length)`
Where A and B are the far and near side measurements of the 5th offcut strip. Result is in millimetres.

**Inches to millimetres:**
`mm = inches × 25.4`

## Tech Stack

- Kotlin
- Jetpack Compose

## Getting Started

Clone the repository and open in Android Studio. Run on an emulator or physical device — no API keys or permissions required.
